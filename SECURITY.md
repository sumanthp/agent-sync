# Security Policy

## Supported Versions
We provide security updates for the following versions of `agentsync-vcs`:

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability
We take the security of `agentsync-vcs` seriously. If you believe you have found a security vulnerability, please do not open a public issue. Instead, please report it privately to our security team.

### How to Report
Please send an email to `sumanthpolisetty.sp@gmail.com` with the following information:
- Description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact if exploited.

We will acknowledge your report within 48 hours and provide a timeline for resolution.
