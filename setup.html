Getting Started

1 Prerequisites

Ensure you have the following installed on your machine:

  • Python 3.8+ (Required for the compiler core)
  • Go 1.21+ (Required to build the CLI binary)
  • Git (To clone the repository)

2 Installation

Clone the repository and install the Python dependencies:

# Clone the repo
git clone https://github.com/sumanthp/agent-sync.git
cd agent-sync

# Install Python requirements
pip install -r requirements.txt

3 Build the CLI

Compile the Go source code into a portable binary:

# Build for your OS (generates agent-sync or agent-sync.exe)
go build -o agent-sync ./cmd/agent-sync

Pro Tip: Move the agent-sync binary to your system PATH to use it from any directory.

4 Initialize Your Project

Navigate to your project root and run the initialization command:

$ agent-sync init

This creates a sample-rule.md and a .agent-sync/config.json file.

5 Connect Remote Rules (Optional)

If your team has a shared repository for agent rules, add it as a remote:

$ agent-sync remote add https://github.com/my-org/agent-rules.git
$ agent-sync sync

The sync command clones the repository to a local cache on your machine.

6 Sync to Your Tools

Use the pull command followed by your target tool. This will compile both local and remote rules:

# For Cursor
$ agent-sync pull cursor

# For Claude Code
$ agent-sync pull claude
Important: agent-sync uses the Python runtime to execute the compiler. Ensure python is available in your terminal's environment.
